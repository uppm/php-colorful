# PHP-Colorful! `UPPM`
```php
echo Colors::BLUE."Hello world!".Colors::ENDC;
// Logging outputter
Colors::error("File not found :(!");
```

#### Installation
Download UPPM if you havent
```bash
wget https://raw.githubusercontent.com/interaapps/uppm/master/uppm
```

Installing colorful!
```php
php uppm install colorful
```
